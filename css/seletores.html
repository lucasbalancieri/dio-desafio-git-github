Seletores CSS

O CSS possui os seguintes seletores:

Seletores:

  • Seletor por Tipo: Busca elementos por uma tag HTML especifica
  • Seletor por ID (#): Busca os elementos através do atributo id
  • Seletor por Classes (.): Busca os elementos através do atributo class
  • Seletor por Classes (.): linha utilizada como exemplo
  • Seletor Universal (*): Selecionta todos os elementos HTML
  • Seletor de Atributo([atrib]): Seleciona os elementos que possuem um atributo especifico no documento HTML ou que possuem atributos com um valor especifico
