Exemlo CSS In Line

Implementação In Line:
Não é muito utilizado, apenas em contextos especificos
Possui prioridade (override) sobre as outras formas de implementação

Implementação Interna

Implementação Interna:
Implementalção feita dentro da tag head;
O código CSS fica restrito à pagina;
Será recarregado toda vez que a página html for recarregada;

Implementação Externa

Implementação Externa:
Maneira mais utilizada;
Boa pratica: arquivos css dentro do diretório assets/css;
Utiliza-se a tag link dentro da tag head para "linkar" o style.css com o html;
