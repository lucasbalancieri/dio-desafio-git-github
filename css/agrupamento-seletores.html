Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, Podemos separar os seletores utilizando vírgula (,)

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, podemos fazer um só bloco de CSS e reutilizar as mesmas regras.
